# AI Resume Skill Recommender

This project recommends new skills based on the skills present in a user's resume.

It also analyzes uploaded resumes and suggests missing skills.

## Features

- Skill recommendation using Machine Learning
- Resume PDF analysis
- Career path prediction
- Interactive UI using Gradio

## Tech Stack

- Python
- Scikit-learn
- Gradio
- PyPDF2

## Example

Input Skills:
Python, HTML, CSS

Output:
Suggested Career Path: Web Developer

Recommended Skills:
- React
- NodeJS
- MongoDB
